### 시험기간 제외 매주 과제가 주어질 예정입니다.
과제 기한은 일요일 23:59까지입니다. <br>
ex)월요일 18:00 ~ 20:00 동아리 활동 <br>
-> 금주 일요일 23:59까지 과제 인정 <br>
